# Jarvis Lab Dashboard
## v1.0 - Live Dashboard

Agent monitoring for Jarvis ops.

## Live Demo
https://jarvisagent001-ui.github.io/the-lab

## Features
- Live polling OpenClaw status (sessions, subagents, models, tokens).
- Green matrix theme.
- Responsive table.
- Local: python3 -m http.server 3000

## Setup (Remote)
Open in browser. API polls localhost:18789/status (local OpenClaw).

## Customization
Edit index.html JS fetch URL.
